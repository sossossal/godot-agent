# 更新日志

## [1.2.0] - 2026-03-24
### 增加
- 引入 `Task` 编排系统,支持复杂指令拆解。
- 引入回滚机制,修改文件前自动备份至 `logs/backups`。
- 引入 Godot 插件实时联动 (Sync Mode),支持不重启注入节点。
- 正式化 CLI 子命令系统 (`run`, `plan`, `doctor`, `roles`, `history`)。
- 增加 `doctor` 环境诊断工具。
- 增加 AI 行为模板库 (巡逻、追击、状态机)。

### 修复
- 修复 Windows 下的 GBK 编码冲突问题。
- 修复 Godot 路径缺失导致的初始化崩溃。
- 修正 API 服务器的启动链路。

## [1.1.0] - 2026-03-24
### 增加
- 统一任务数据模型 (Task / Artifact / ToolResult)。
- 实现代码落盘功能。
- 增加基础场景创建模板。

## [1.0.0] - 初始版本
- 基础五角色架构。
- 基础 Godot CLI 调用。
